HTML and layout CSS for site pages

This week we're going to start building our webpages. For each layout on your site, make an HTML page with content and the basic CSS layout. We will work on styles next week.

    Your HTML pages should have:
  • Basic HTML structure.
  • Meta content, including title, charset, stylesheet, keywords, description, and anything else you need as meta content.
  • All of the layout divs and sections that are needed, using either divs with ids, or HTML5 semantic tags.
  • Dummy content for text and media. (If your real content is ready, use that.)
  • Comments.
  • Validation. Record screenshots of your validation, or the error reports you get and explain how you fixed them.
    CSS pages should include:
  • Any declarations you need, like @charset, @font, or @media.
  • A container div.
  • Layout divs with width & height, position and other layout properties.
  • Resets.

Next week we will start making styles and working with transitions.
