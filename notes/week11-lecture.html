Introduction to jQuery

What is jQuery?

jQuery is a JavaScript library that simplifies scripting for HTML pages using DOM element selectors and a set of animations, transitions and other manipulations based on CSS and HTML. DOM refers to the Document Object Model, which is a cross-platform set of conventions for interacting with objects in HTML, XML and XHTML documents through scripting languages. By giving direct access to the DOM, jQuery allows us to write JavaScript that easily traverses the hierarchy of a page to manipulate specific elements of our markup, by simply changing CSS attributes, adding animations, or doing more complex scripting. jQuery was released in 2006 and has become extremely popular with web design. One of the features of jQuery is that it is compatible with all major browsers.

So what is JavaScript?

We'll talk about this more next week, and next semester. But briefly, JavaScript is a scripting language for browsers that began to be developed at Netscape in the early 90s and later introduced to Microsoft and other browsers. While JavaScript was intended to provide the functionality of the language Java, it's syntax more closely resembles the C programming language. It was meant to be a high level language aimed at non-professional programmers, like Visual Basic, or something similar to the relationship between Processing and Java. It was made to implement client-side interactions between browser and user but since has been used to develop server-side programs and applications. There's a long and interesting history behind JavaScript that we won't get into now, but check out the lecture by Douglas Crockford linked in the resources section if you want to learn more.

So how do we use jQuery?

jQuery is a file that is attached to the document, similar to a CSS file. You can host your own copy of jQuery, which you can download from their site or several hosts, and then link to, or you can download a copy hosted on another server, like Google. It uses the script tag.

<script type="text/javascript" src="jquery.js"></script>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

Once you have linked the jQuery file, you can begin writing scripts that use jQuery, in a script tag, either in the <head> section or <body>.

<script type="text/javascript">
    // 
    // jQuery goes here
    //
  </script>

Or you can write jQuery into a separate file and link it like the jQuery library.

<script type="text/javascript" src="myjQueryFile.js"></script>

jQuery uses a $ to define and access the library. The basic syntax of a jQuery function looks like this: $('selector').action(). Like in CSS, the selector refers to an HTML element. The action refers to the script that will be executed on that element. The rest of the syntax comes from JavaScript. For today we will do simple jQuery examples. Next week we will look at some basic programming concepts like data types, conditional statements and variables, which will help us do more complex operations using jQuery.

Let's look at a simple example.

$(document).ready(function() {
  $('myDiv').click(function() {
    $(this).css('color', 'blue');
  });
});
Click me!

In this example, the document refers to the HTML page being loaded, and ready is an event that will call the function written inside the parentheses (the argument). This is used so that the function will not be called before the document is completely loaded, which might cause problems if actions are called on elements that have not yet been loaded.

Once the document loads, it will call the function we write as the argument. This function calls jQuery with the $ and then finds the id called "myDiv". This would also work for a class or a generic HTML tag. Then it uses a click event to call a function when that HTML element is clicked. In that function, this refers to the element that has launched the click event. this or #myDiv is then assigned a new CSS value for the color property. You can replace the property and value with anything that is compatible with CSS.

jQuery can be used to make elements appear and disappear:

  $(document).ready(function() {
    $('#hideButton').click(function() {
      $('#mygraf').hide();
    });
    $('#showButton').click(function(){
      $('#mygraf').show();
    }); 
  });

My paragraph.

Or you can toggle between the two.

$(document).ready(function() {
  $('#toggle').click(function() {
    $('#newgraf').toggle();
  });
});

My new paragraph.

You can also use a number of different events to launch functions, like dblclick, keypress, scroll and others. Let's use a keypress to hide a div.

$(document).ready(function() {
  $('#key').keypress(function() {
    $(this).toggle();
  });
});

jQuery also has durational effects that can be used for transitions, such as fade, slide and animate. These methods add a couple of arguments to their function call, the speed argument, which determines how long the transition will last, and a callback, which is a function to launch once the transition is finished. Speed is measure in milliseconds, so the argument 1000 is equivalent to 1 second. You can also write 1s. Let's look at a couple of examples.

$(document).ready(function() {
  $('#fade').click(function() {
    $('#redsquare').fadeToggle(300);
  });
});
You can use fadeIn(), fadeOut(), and fadeToggle(), or fadeTo(), which adds an argument for a specific opacity.
$(document).ready(function() {
  $('#slide').click(function() {
    $('#bluesquare').slideToggle(1000);
  });
});
$(document).ready(function() {
  $('#animate').click(function() {
    $('#greensquare').animate({
      fontSize:'+=0.3em',
      left:'100px',
      width:'-=10px'
    });
  });
});
Animate

Another important method is the stop() method, which will stop all or a specific animation. This is a good thing to use with multiple animation states, like mousein and mouseout.

I mentioned callback functions earlier. Let's look at how that would work.

$(document).ready(function() {
  $('#callback').click(function() {
    $('#purple').animate({
      fontSize:'-=1em',
      opacity:'-=.1'
    }, 1000, function() {
      alert('I\'m fading away.');
    } );
  });
});
I'm shrinking.

You can also "chain" jQuery actions together.

$(document).ready(function() {
    $('#chain').click(function() {
      $('#orange').css('color','blue')
                  .slideUp(500)
                  .slideDown(100);
  });
});
I'm a chain.

jQuery can also be used to edit the HTML iin a document. You can get values from HTML, set values, add content or markup, remove content or markup and changes CSS attributes. Let's look at a couple of examples.

$(document).ready(function() {
    $('#get').click(function() {
      alert('This elements says: ' 
              + $('#lightblue').text());
    });
    $('#set').click(function() {
      $('#lightblue').text('I\'m new HTML content.');
    });
    $('#add').click(function() {
      $('#lightblue').append('This is added content.');
    });
    $('#empty').click(function() {
      $('#lightblue').empty();
    });
});
I'm some HTML.

You can also move around the HTML document using parent, child, sibling, descendant and ancestor relationships.

$(document).ready(function() {
  $('#child').click(function() {
    $('#lightgreen').children().css('color','blue');
  });
});

Or you can find specific elements.

$(document).ready(function() {
  $('#find').click(function() {
    $('#lavender').find('p').css('color','hotpink');
  });
});
