Navigation menu

Today we're going to make a navigation panel using the CSS box model, CSS floats and psuedo classes. Some of these concepts are new and some we've talked about, but they are all going to be essential to design and layout going forward.

Most site navigation starts with an unordered list.

<ul><li>Home link</li> <li>About page</li> <li>Blog page</li> <li>Etc</li> </ul>

The first thing we want to do is get rid of those bullet points. To do this we have to set the <li> to list-style: none;. But first, let's create a 'nav' id tag in CSS so that our styles will only apply to our menu, and not all the other lists littering our webpages: #nav { }. Depending on whether you want a vertical or horizontal menu, you may want to then float your <li> (list items).

Once we've chosen our orientation, let's work with the CSS box model to get the right spacing between our elements to make the navigation menu visible and clear.
