Fixed position drop down nav menu

Example.

Fixed position elements are position relative to the browser window. We don't use them very often while laying out a basic page, but they can be useful for notifications, headers and navigation menus. We will build a fixed position navigation menu like the one in the example above.

The example also shows a way to use the display properties of CSS to create a drop down effect to accommodate multiple navigation options under each menu item.
