Final project proposal

Next week we will give presentations during the lecture section of class. Each student should have a presentation prepared using a slideshow or powerpoint (or web based slide show) about the site you want to make as a final project over the next several weeks. You should present the content, functionality and some preliminary design concepts.

The presentation should include:

  • Overall pitch for site content.
  • Overview of functionality: number of pages, types of interactions and menus, intended user experience.
  • Media plan for images, video and text content.
  • Site architecture diagram.
  • Examples of inspiration sites.

The presentations should be five to ten minutes, followed by five minutes of feedback.

Don't forget to choose a website for the hallway screen. Take some time to fix any errors and clean up the design.
