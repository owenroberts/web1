Create an interactive animation

Read Computers, Pencils and Brushes by Paul Rand

Using JavaScript and CSS3, create an interactive animation. You could make a graphic, a navigation menu, or a part of a website, or something more abstract. Find some examples of animated designs to use for inspiration, I recommend checking out the infographics made by The New York Times or looking at things like this list of JavaScript/CSS3 projects on Smashing Coding and other design blogs.
