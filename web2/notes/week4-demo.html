JavaScript and CSS3

Last week we reviesed HTML and CSS. I want to look at a new property of CSS before we start using JavaScript libraries like jQuery. CSS3 has a transition property that allows us to do a lot of things that formerly required more JavaScript or jQuery to do, with just CSS and a little JavaScript. We can use CSS to make transitions between colors, sizes and positions of our elements, using little or no scripting. Not all CSS properties can have transitions applied to them, but most properties for color, size and position can be animated. In many cases, CSS psuedo states like :hover or :active can be used to trigger transitions, but we can also set up transitions to be triggered by simple JavaScript changes. Look at this simple example:

<style>
p#ex1 {
  transition: color 2s;
}
</style>
<p id="ex1">Watch the color change.</p>
<button onclick='document.getElementById("ex1").style.color = "blue";'>
  Click</button>

Watch the color change.

I didn't even need a script tag for that one. The transition property requires a property to transition and a duration. We can write durastion in seconds or milliseconds, ie: 300ms would be 300 milliseconds. We can also use all to apply transitions to any changes to the CSS style of an element, but those styles must be specified.

<style>
  p#ex2 {
    position:relative;
    width:200px;
    height:50px;
    left:0;
    transition: all 3s;
  }
</style>
<p id="ex2">Multiple changes.</p>
<button onclick='allChange();'>Click</button>
<script>
  function allChange() {
    var p = document.getElementById('ex2');
    p.style.color = "white";
    p.style.background = "blue"
    p.style.width = "300px";
    p.style.height = "200px";
    p.style.left = "50px";
  }
</script>

Multiple changes.

A third value can be added to the transition property to determine the style of animation with the transition timing property, including easings or steps.

<style>
  p#ex3 {
    position:relative;
    background:purple;
    width:200px;
    height:50px;
    left:0;
    transition: all 3s ease;
  }
  p#ex4 {
    position:relative;
    background:blue;
    width:200px;
    height:50px;
    left:0;
    transition: all 3s steps(5, end);
  }
</style>
<p id="ex3">Easing</p>
<p id="ex4">Stepping</p>
<button onclick='ease();'>Click</button>
<button onclick='step();'>Click</button>
<script>
  function ease() {
    var p = document.getElementById('ex3');
    p.style.left = "400px";
  }
  function step() {
    var p = document.getElementById('ex4');
    p.style.left = "400px";
  }
</script>

Stepping

Besides being used for easier transitions in simple design elements like hyperlinks, navigation bars or slideshows, these are powerful methods for interactive graphics.
