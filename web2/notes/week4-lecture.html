Logic and loops

Statements

Statements are blocks of code that control the flow of operations and expressions to determine interactivity in a webpage. Expressions and operations are statements. Statements are separated by semi-colons. Conditional logic and loops are two statements that allow us to add interactivity to a page, by creating rules and conditions for the script to act out. We're going to look at logic and loops today.

Logic

Conditional statements are actions that are executed if a tested condition is true. JavaScript uses to formats for conditional statements: if ... else and switch. Here's an example of a conditional statement:

function input() {
    var color = prompt("Input 1 or 2");
    if (color == 1) {
      document.getElementById('if').style.background = "red";
    } else if (color == 2) {
      document.getElementById('if').style.background = "blue";           
    }
}

Input 1 for red and 2 for blue.

If the user inputs 1, the first condition will be true. If the user inputs 2 the second condition will be true. Otherwise no condition will be satisfied and nothing will happen. else if could be changed to else which would execute as long as the first condition doesn't evaluate to true. You can nest conditional statements and test for as many conditions as desired.

Last week we talked about booleans and all the values that evaluate to false, including false, undefined, null, 0, NaN and empty string, "".

JavaScript uses the == to determine if values are equivalent, resulting in an evaluation of true of false. The != operator returns true if two values are not equivalent, or ! before a value to indicate the opposite, ie: !true is false. == ignores data type, so 7 will be equal to "7". The === sign evaluates value and data type, so 7 === "7" will return false. There are a number of logical statements that help us evaluate more complex statements. We can use && and || to add statements to our condition. && requires all statements to evaluate true, reading left to right. || requires one of the statements on either side to be true. Try these example in the console:

true && false
 7 == "7" && NaN != 1
 false || true
  7 === "7" || 1 == 1 && !(false)

We can also uses parathesis () to nest logic rules.

(true&&true||false) && true

switch statements evaluates an expression and matches that with a series of cases to determine which action to perform.

function switcher() {
  var s = prompt('Input number.');
  var p = document.getElementById('switch');
  switch (s) {
    case "1":
      p.style.color = "blue";
      p.style.background = "lightblue";
      break;
    case "2":
      p.style.color = "green";
      p.style.background = "lightgreen";
      p.style.fontSize = "100px";
      break;
    case "3":
      p.style.fontSize = "9px";
      p.style.background = "purple";      
  }
}

Enter a number between 1 and 3.

Loops

A loop is a set of commands that is executed repeatedly until a condition changes. JavaScript uses several loops statements. The most common is possibly the for statement.

for (var i = 0; i < 5; i++) {
  console.log("This is the " + i + " loop.");
}

Looking at that piece of code, the for keyword indicates the beginning of the for loop. Inside the parentheses there are three statements that control the duration of the loop. The first statement establishes the initial condition: var i = 0;. The second statement establishes the end condition: i < 5;. As long as this condition evaluates to true, the loop will continue. The final statement executes at the end of each loop to change the condition: i++;. The code block for the loop is then contained in the curly brackets: { ... }.

Loops are an efficient way to excecute actions on a list or array of data.

var imgs = ["cat.jpg", "dog.jpg", "bird.jpg"];
for (var i = 0, len = imgs.length; i < len; i++) { 
  console.log('<img src="'+imgs[i]+'" >');
}

A do ... while statement repeats until a condition evaluates to false.

i = 0;
do {
  i += 1;
  console.log(i);
} while (i < 10);

A while statements executes as long as the first statement is true.

i = 0;
while (i < 5) {
  i++;
  console.log("Loop " + i);
}

JavaScript also uses loops to manipulate objects, such as for ... in. We'll talk more about objects next week.
