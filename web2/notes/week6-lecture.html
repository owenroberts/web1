Repetition vs. random

Repetition is one of the fundamental elements of design. Using a shape more than once is repetition. SVG is a convenient way for us to do repetitive designs because they are small and mutable. We can use the SVG graphics we just learned along with our for loops to create multiple copies of a shape.

Example 1

var container = document.getElementById('example1');
var s = document.getElementById('s1');
for (var i = 0; i < 10; i++) {
  var clone = s.cloneNode(s);
  container.appendChild(clone);
}

We can also create new SVGs in Javascript and then append them to the document.

Example 2

var container = document.getElementById('example2');      
for (var i = 0; i < 10; i++) {
  var s = '<svg id="s1" width="100" height="100">
    <circle cx="50%" cy="50%" r="40" fill="blue" stroke="green" stroke-width="10" />
  </svg>';
  container.innerHTML += s;
}

The advantage of using SVG is that we can create shapes that are not boxes or rectangles. This is possible with HTML elements, but it requires hacking the usual layout of the elements. We can easily create SVGs of any shape in code or design them in Illustrator and use them on a webpage.

Example 3

var container = document.getElementById('example3');
var s = document.getElementById('s3');
for (var i = 0; i < 10; i++) {
  var clone = s.cloneNode(s);
  container.appendChild(clone);
}

When we use repetition with a variety of elements we create patterns. We can use the same for loop to make patterns with SVG elements, or we can use the SVG pattern definition, which allows us to define a visual block and then use it as a repeating pattern. These are easy to implement and modify.

We can make more complex patterns by varying shape, size and color.

We can vary our pattern by using a for loop with different computations.

Example 6

var c = document.getElementById('example6');
var s = [
  document.getElementById('p1'),
  document.getElementById('p2'),
  document.getElementById('p3'),
  document.getElementById('p4')
];
for (var i = 0; i < 64; i++) {
  var clone = s[i%s.length].cloneNode(s[i%s.length]);
  c.appendChild(clone);
}

Let's modify it with a simple ternary switch.

Example 7

var c = document.getElementById('example7');
var s = [
  document.getElementById('p1'),
  document.getElementById('p2'),
  document.getElementById('p3'),
  document.getElementById('p4')
];
var sw = false;
for (var i = 0; i < 64; i++) {
  var n = i%4 + (sw ? -1 : 1);
  var clone = s[n].cloneNode(s[n]);
  c.appendChild(clone);
  sw = !sw;
}

Or run through the shapes backwards.

Example 8

var c = document.getElementById('example8');
var s = [
  document.getElementById('p1'),
  document.getElementById('p2'),
  document.getElementById('p3'),
  document.getElementById('p4')
];
var sw = false;
for (var i = 0; i < 64; i++) {
  var n = 3 - i%4;
  var clone = s[n].cloneNode(s[n]);
  c.appendChild(clone);
  sw = !sw;
}

Randomness is a way to add variability to a design using math and the computer. Take a look at the Math functions available in Javascript, including Math.random(). Let's also take a look at how randomness can be used in design with the MIT generative logo.

Example 9

function getRandomNumber(n) {
  return Math.floor(Math.random()*n);
}
var c = document.getElementById('example9');
var s = [
  document.getElementById('p1'),
  document.getElementById('p2'),
  document.getElementById('p3'),
  document.getElementById('p4')
];
for (var i = 0; i < 132; i++) {
  var n = getRandomNumber(s.length);
  var clone = s[n].cloneNode(s[n]);
  c.appendChild(clone);
}

We can also use our own Illustrator drawings with code to make patterns. This SVG could easily have been created in code, but it might be easier to design and modify in Illustrator. The file can then be loaded into an img, object or embed tag.

Example 10

for (var i = 0; i < 24; i++) {

  img.src = "../demo/svg/green-zag.svg";
  img.style.width = "80px";
  img.style.height = "80px";
