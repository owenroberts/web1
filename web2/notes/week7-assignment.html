Visualize a data set with user input

Create a visualization of a simple set of data (the olympic medal count projects from week 3 are a good example) and create a design that can be manipulated by a user. Use shape, size and color to represent different qualities of the data. The user can click on or hover over different part of the graphics or enter data, it doesn't necessarily need to use basic forms. This project should combine the user input stuff we learned at the beginning of the semester with some of the graphics, repetition and animation we've learned recently.

Research a JavaScript library

We've looked at two JavaScript libraries recently, jQuery and d3. There are hundreds more which can provide special functionality to our websites. Do research on one library, look at the examples of how it is used and talk about what you might want to use it for. Look at the documentation and take note of some of the methods the library uses. Start thinking about what you might want to do as a final project so you can consider using a specific library (though this isn't required that the library is used in the final). Prepare a short (2-3 minute) presentation to show some example and talk about the general idea behind the library.
