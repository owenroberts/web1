JSON stands for JavaScript Object Notation. It is a lightweight data format that is easy to write and easy for JavaScript to parse. Many of the examples we have looked at this semester use JavaScript objects, which the syntax for JSON is based on.

I made an example based on the Open Movie Database API which is an API for film data, unaffiliated with the IMDb. See what you get when you type in the following to your URL bar:

http://www.omdbapi.com/?t=Space Jam

We've seen that before right? Let's try assigning it to a JavaScript variable, using jQuery.

$.getJSON('http://www.omdbapi.com/?t=Space Jam', function(json){  
  console.log(json);
});

In the demo linked above, we can then use this data to make a simple web app or populate some information.
